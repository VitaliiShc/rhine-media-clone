# Rhine Media Clone

Frontend test task based on the Rhine Media landing page.

## Tech Stack

- React
- Vite
- JavaScript
- CSS Modules
- HTML5
- Mobile First approach

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
